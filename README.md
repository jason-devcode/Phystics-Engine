# PhysTics Engine

Este es un motor de físicas desarrollado con fines educativos donde el objetivo principal es aprender sobre física de una forma lúdica y creativa utilizando razonamiento matemático puro y programación manual tradicional.

## Tecnologías utilizadas
1. Lenguaje de programación C versión C23
2. Para la visualización de los frames calculados se usara SDL 2

El motor busca ser desarrollado agnóstico a frameworks o librerías para calculos especializados por lo que las únicas dependencias serán las propias librerías ofrecidas por el estándar ANSI C y el manejo de ventanas y eventos con SDL2.
